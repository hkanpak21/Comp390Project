## What §7 of the paper needs from this run

`paper/sections/07_end_to_end.md` needs three data points extracted from
stdout (one per trial):

1. **t_layer_1**: reported per-trial execution time for layer 1.
2. **t_layer_2**: reported per-trial execution time for layer 2.
3. **Saturation check**: compute `delta = |t_2 - t_1| / t_1` for median trial.
   - **Pass** iff `delta <= 0.05` (5% threshold as per PRD).
   - **Fail** triggers re-investigation of chain composition or per-head timing.

**Extrapolation to full BERT:**
- `t_per_head_per_layer = mean(t_layer_1, t_layer_2)` for the median trial.
- Full BERT (12 heads × 12 layers) ≈ `144 × t_per_head_per_layer`.
- This extrapolation is reported in §7 only if saturation check **passes**.

The post-run BACKFILL-S7 slice will:
1. Extract t_layer_1 and t_layer_2 from the median trial's stdout.
2. Apply MEASURE-02 (saturation analyzer) to compute delta and pass/fail.
